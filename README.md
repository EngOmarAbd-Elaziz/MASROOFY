## 🎬 Overview

MASROOFY is a modern desktop finance management system designed to help users take full control of their personal finances in a simple, visual, and structured way.

It allows users to track income, manage expenses, define budgets, and analyze spending behavior through an interactive dashboard.

The project is built as a real-world production-style application, focusing on clean architecture, usability, and financial clarity.


---

## 🎯 Core Idea

Most people lose track of their money because:

Expenses are scattered

No clear budget system exists

No visualization of spending habits


👉 MASROOFY solves this by turning finance into a simple flow.


---

## ⚡ Core Flow (User Journey)

`Add Income → Add Expenses → Set Budget → View Dashboard → Analyze Spending → Improve Financial Habits`


---

## 👤 User Story

As a user, I want to:

Track my daily income and expenses

Understand where my money goes

Set monthly budgets for control

View visual analytics of my spending

Improve my financial habits over time



---

## 🧠 Use Cases

> 💰 Income Management

  Add income sources

  Edit or delete income entries


> 💸 Expense Management

  Add categorized expenses

  Track spending history

  Filter by category or date


> 🎯 Budget System

  Set monthly budget limits

  Compare actual vs planned spending


> 📊 Analytics Dashboard

  Visual charts (Pie / Bar)

  Spending breakdown by category

  Income vs expenses comparison

## 🏗️ System Architecture

_MASROOFY is built using a clean Layered Architecture:_

 `Presentation Layer (UI) → Business Logic Layer → Data Access Layer → Database Layer`

> 🔹 Design Principles:

    1. Object-Oriented Programming (OOP)

    2. Separation of Concerns

    3. Reusable Services

    4. Scalable Structure
